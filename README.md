# Buscador de CEP - Java
Aplicação em Java para consulta de CEP utilizando API ViaCep.

<h2> Como funciona 🎮 </h2>
<ul>
  <li> Consulta de CEP via API. </li>
  <li> Mensagem de erro caso CEP esteja inválido. </li>
  <li> Geração de arquivo JSON após dados consultados. </li>
  <li> Exibição de dados no console. </li>
</ul>

## Tecnologias 🚀 
 <ul>
  <li> Java. </li>
  <li> API REST. </li>
  <li> HTTP Client. </li>
  <li> Gson (manipulação de JSON). </li>
</ul>

## Como executar ▶️
 <ul>
   <li> Abrir o projeto em uma IDE de sua preferência. </li>
   <li> Executar a classe principal. </li>
   <li> Inserir o CEP desejado no console. </li>
   <li> Visualizar os dados retornados. </li>
 </ul>
